| **Descripcion**            | VARCHAR    | 200 caracteres   | NOT NULL          | -           | -           | Descripción detallada del recurso                  |

| **Rastreo_inventario**     | BOOLEAN    | -                | `TRUE`, `FALSE`   | -           | -           | Indica si el recurso es rastreable en inventario   |
| **Costo_unitario**         | NUMERIC    | (10,2)           | > 0               | -           | -           | Costo unitario del recurso                         |
| **Peso**                   | NUMERIC    | (7,2)            | > 0               | kg          | -           | Peso del recurso                                   |
| **Volumen**                | NUMERIC    | (7,2)            | > 0               | m³          | -           | Volumen del recurso                                |
| **Stock**                  | INT        | 99999            | > 0               | unidades    | -           | Cantidad total disponible en inventario            |
| **Disponible**             | INT        | 99999            | > 0               | unidades    | -           | Cantidad disponible para uso                       |
| **Minimo**                 | INT        | 99999            | > 0               | unidades    | -           | Cantidad mínima requerida para reabastecer         |
| **Maximo**                 | INT        | 99999            | > 0               | unidades    | -           | Cantidad máxima permitida en inventario            |
| **Pedido**                 | INT        | 99999            | >= 0              | unidades    | -           | Cantidad pendiente de pedido                       |
| **Plazo_entrega**          | INT        | 99999            | > 0               | días        | -           | Plazo de entrega estimado                          |
| **Cod_estado_recurso**     | CHAR       | 1 carácter       | NOT NULL          | -           | -           | Estado del recurso (por ejemplo, "Disponible", "No disponible") |
| **Fecha_registro**         | DATE       | -                | NOT NULL          | -           | -           | Fecha de registro del recurso en el sistema        |

| **ID_lote**                | CHAR       | 6 caracteres     | NOT NULL          | -           | -           | Identificador del lote del recurso                 |

# 7.1. Creación de Tablas

##  Procedimiento de Creacion de Tablas
Se utilizó un modelo Entidad Relación previamente diseñado en **ERDPlus** y lo tomamos de guía para rehacer todo el modelo Entidad Relación en **LucidChart**, debido a que podiamos trabajar simultáneamente en la plataforma, cosa que ERDPlus no posee, luego de quedar conformes con el modelo actual, uno de nosotros pasó toda la información del LucidChart al ERDPLus, con el apoyo de los compañeros, para posteriormente convertir el modelo a un modelo lógico, en el cual reacomodamos las entidades y empezamos a editar cada atributo de cada entidad para que esté con el tipo de dato correcto. Finalmente convertimos el modelo lógico a un Script SQL con la herramienta ERDPlus que ayudó a generarlo automaticamente.
El script SQL generado fue revisado y comparado con los códigos de tablas que cada miembro del equipo había realizado por adelantado, realizándose modificaciones manuales en las tablas para corregir y mejorar la estructura.
Después de realizar las correcciones, se ejecutó el código de creación de tablas en la base de datos y se verificó que todas las tablas se crearán correctamente.
